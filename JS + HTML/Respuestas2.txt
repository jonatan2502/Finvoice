3. ¿Qué significa HTML, y cuál es su estructura básica?
Es un lenguaje de marcado que se utiliza para la creación de páginas web. Se compone básicamente de una etiqueta <html> que envuelve todo el documento, dentro se encuentra la etiqueta <head> en la cual se incluye el título, enlaces a scripts, hojas de estilos y cualquier otro tipo de metadatos. Le sigue la etiqueta <body>, como su nombre lo indica, representa el cuerpo del documento, en el se incluyen todos los elementos que se quieren mostrar; algunos de estos elementos son divs (contenedores), imágenes, párrafos, formularios, entre muchos más.
